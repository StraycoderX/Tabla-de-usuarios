# Tabla de usuarios

Aplicación en **Python** que implementa una tabla de usuarios dinámica con funcionalidad **CRUD**
completa (Crear, Leer, Actualizar y Eliminar registros).

## ✨ Características

- Crear nuevos usuarios.
- Listar y consultar usuarios existentes.
- Actualizar los datos de un usuario.
- Eliminar usuarios.

## 🛠️ Requisitos

- Python 3.10 o superior.

## 🚀 Uso

```bash
# Clonar el repositorio
git clone https://github.com/StraycoderX/Tabla-de-usuarios.git
cd Tabla-de-usuarios

# Ejecutar la aplicación
python "jemlo crud2.py"
```
